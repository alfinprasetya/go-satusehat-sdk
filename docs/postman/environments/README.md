# Postman Environments

Place non-secret Postman environment templates in this directory.

Recommended naming:

- `satusehat-staging.postman_environment.json`
- `satusehat-production.postman_environment.json`

Environment files committed to this repository must only contain variable names and safe placeholder values. Do not commit client IDs, client secrets, organization IDs, bearer tokens, refresh tokens, NIK values, or production patient data.
