# Postman Collections

This directory contains SATUSEHAT FHIR Postman collection exports used as SDK implementation guidelines.

Current collection:

- `00. FHIR Resource - Contoh Penggunaan.postman_collection.json`

Official source:

- [00. FHIR Resource - Contoh Penggunaan](https://www.postman.com/satusehat/satusehat-public/collection/u2k8uiz/00-fhir-resource-contoh-penggunaan)
