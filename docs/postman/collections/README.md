# Postman Collections

Place SATUSEHAT FHIR Postman collection exports in this directory.

Official source:

- [00. FHIR Resource - Contoh Penggunaan](https://www.postman.com/satusehat/satusehat-public/collection/u2k8uiz/00-fhir-resource-contoh-penggunaan)

Recommended naming:

- `satusehat-fhir-r4.postman_collection.json`
- `satusehat-patient.postman_collection.json`
- `satusehat-encounter.postman_collection.json`

Collections should act as source guidelines for SDK endpoints, resource payloads, and examples. Prefer collections from official SATUSEHAT documentation, and keep any manual edits focused on readability or grouping.
